.. _Glossary:

Glossary
================================================================================================================

.. glossary::

    Applicants
        People whose information is stored in this Database

    Candidates
        People whose information is stored in this Database

    OCR
        Optical Character Recognition

    User
        Another term for members of the Recruitment Team

    Authentication
        It is the process of recognizing a user's identity.

    Database
        It is an organized collection of structured information, or data, typically stored electronically in a computer system.

    File System
        It is a method and data structure that the operating system uses to control how data is stored and retrieved.

    Recruitment Tool
        It is an advertising method that aids in creating interest in and getting people for a typically political organization.

    Retrieve Data
        It is the process of identifying and extracting data from a database, based on a query provided by the user or application.

    Search Engine
        It is a program that searches for and identifies items in a database that correspond to keywords or characters specified by
        the user, used especially for finding particular sites on the World Wide Web.

    Storing Data
        It is the process of recording information in a storage medium.

    User Interface
        It is the point of interaction between the user and the product they are using. So, UI design focuses on all the visual
        and interactive elements of a product interface, such as buttons, menu bars, icons, and more.

    Web Application
        It is the software that runs on a web server, unlike computer-based software programs that are run locally on the operating system of the device.

    Wireframe
        It is a two-dimensional illustration of a page's interface that specifically focuses on space allocation
        and prioritization of content, functionalities available, and intended behaviors.
